# Contributing

Loom (the implementation tier) is in founding-draft (Preview) stage. The
highest-value contributions sharpen the mode-contract seam or add a reference
component's contract against it.

## What to contribute now
- **Mode-contract review** — the three-surface seam (§3) is young. Feedback on
  whether the surfaces are complete (can every component be driven through only
  these?) is the most valuable input.
- **Reference component contracts** — a new component's mode-contract, checked for
  consistency against its core machine.
- **Adapter-fidelity checklist** — refinements to what a thin adapter may/‑may‑not
  do (§5).

## Conventions
- `CHANGELOG.md` uses `## Planned`, never `[Unreleased]`.
- Naming: `loom-<platform>` = mode; `loom-<platform>-<framework>` = adapter.
